## ğŸš€ Upgrade Guide

### Docker Users

```bash
# Stop current container
docker-compose down

# Pull new version
docker pull ghcr.io/rokk001/cineripr:2.0.0
# or
docker pull ghcr.io/rokk001/cineripr:latest

# Start updated container
docker-compose up -d
```

**No configuration changes needed!**

### pip Users

```bash
# Upgrade package
pip install --upgrade cineripr

# Verify version
cineripr --version
# Should show: 2.0.0
```

**Your `cineripr.toml` works without changes!**

### Developers

```bash
# Pull latest code
git pull origin main

# Update dependencies
pip install -e .[dev]

# Update imports in your code
# (see Breaking Changes section)

# Run tests
./scripts/run-tests.sh
```
